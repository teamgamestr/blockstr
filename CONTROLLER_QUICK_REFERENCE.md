# Controller Quick Reference Guide

## üéÆ Blockstr - USB Controller Controls

### Start Screen
- **D-Pad Up/Down** or **Left Stick**: Navigate options
- **A Button**: Confirm selection
- **Start Button**: Quick start

### How To Play Screen
- **A Button**: Start game
- **Start Button**: Start game

### During Gameplay
| Action | Controller Button |
|--------|------------------|
| Move Left | D-Pad Left / Left Stick / LB |
| Move Right | D-Pad Right / Right Stick / RB |
| Rotate | D-Pad Up / Left Stick Up / A / X |
| Hard Drop | D-Pad Down / Left Stick Down / B |
| Pause | Start Button |

### Game Over Screen
- **D-Pad Up/Down** or **Left Stick**: Navigate options
- **A Button**: Select option
- **B Button**: Close modal

Available options:
1. Save Score to Nostr (if logged in)
2. Share Score (after saving)
3. New Game
4. Close

## Tips

üí° **All buttons work everywhere** - You never need to switch to keyboard/mouse

üí° **Focus rings** - Look for the colored ring around buttons to see what's selected

üí° **Multiple controllers** - First connected controller is used automatically

üí° **Hot-plug** - Connect/disconnect controller anytime

üí° **Works with keyboard** - Controller and keyboard work together

## Troubleshooting

‚ùì **Controller not working?**
1. Make sure it's plugged in via USB
2. Check browser console (F12) for "Gamepad connected" message
3. Try pressing A button - you should see focus changes
4. Test at [gamepad-tester.com](https://gamepad-tester.com)

‚ùì **Wrong buttons?**
- Most standard controllers follow Xbox/PlayStation layout
- If buttons are mapped differently, your controller may be non-standard

‚ùì **Laggy or delayed?**
- Close other tabs/applications
- Try a different USB port
- Restart browser
